书写提交代码时请大家注意：

1. .gitignore文件已经配置好了，大家不要修改了。

2. push代码之前一定要执行“git pull --rebase”命令操作，以防代码冲突。

3. push命令用“git push origin HEAD”即可。

4. **图片资源文件** 的命名以“book_”开头，并放在**mipmap-xxhdpi**文件夹中。

5. 自己写的代码，最好在代码内容中注释标记上作者名字，若修改代码找人方便。
